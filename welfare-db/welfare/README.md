# Bundled welfare catalogs

- `national.json` — 중앙(전국) 단위 안내
- `gyeonggi.json` — 경기도
- `yongin.json` — 용인시
- `data-go-kr-filedata.json` — 공공데이터포털 등 CSV URL 자동 수집(매니페스트·Actions; `docs/data-sources.md` 참고)

내용은 **참고용 요약**이며, 자격·금액·기한은 반드시 **각 기관 공식 공고**를 확인하세요.

전국 목록을 공공데이터 API로 갱신하려면 저장소 루트 `docs/data-sources.md`와 `npm run data:welfare:national`을 참고하세요.

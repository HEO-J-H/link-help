# PRD (요약)

Link-Help는 가족 단위 프로필과 태그 기반 필터로 복지 항목을 추천합니다. 가족·프로필은 기본적으로
로컬(sessionStorage)에만 두며, 선택 API 연동 시에도 **복지 메타만** 전송하고 프로필은 보내지 않습니다.

**사회적 목표**: 수익이 아니라, 알지 못해 받지 못하는 복지·혜택을 줄이기 위한 **무료 맞춤 안내**. 그를 위해 **신뢰할 수 있는 복지 카탈로그(DB)**가 핵심입니다. **선택 자가 호스팅 API**(`server/`)로 공고 분석·공용 DB·기여를 붙일 수 있고, 기본 배포는 번들+IndexedDB만 사용합니다. 상세: `docs/catalog-pipeline.md`.

상세 요구는 저장소 루트 `복지 알림 시스템 Link-Help.txt`를 기준으로 합니다.

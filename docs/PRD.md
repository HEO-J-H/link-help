# PRD (요약)

Link-Help는 가족 단위 프로필과 태그 기반 필터로 복지 항목을 추천합니다. 개인정보는 서버로 보내지 않고 로컬에만 저장합니다.

**사회적 목표**: 수익이 아니라, 알지 못해 받지 못하는 복지·혜택을 줄이기 위한 **무료 맞춤 안내**. 그를 위해 **신뢰할 수 있는 복지 카탈로그(DB)**가 핵심이며, 장기적으로는 **공고문 → AI 구조화 → JSON → 공용 DB → 전 사용자 재사용** 파이프라인(중복 제거·출처·검증 포함)을 두는 방향을 문서로 정의합니다. 상세: `docs/catalog-pipeline.md`.

상세 요구는 저장소 루트 `복지 알림 시스템 Link-Help.txt`를 기준으로 합니다.
